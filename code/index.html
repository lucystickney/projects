Terminal Device Driver - C

Developed a device driver that manages keyboard input from a user and output from user processes. The device driver uses a Mesa monitor to ensure that access to the terminals do not cause data races, while still being able to manage multiple terminals at once. The driver uses conditional variables to ensure that multiple processes and the user cannot write to one terminal at the same time.

Knitting Chart - HTML, CSS, JavaScript

Created a website meant for knitters. Fiber artists sometimes use a grid of squares to map out knitted and crocheted patterns, called a chart. The chart is a 10 by 10 grid of squares, and if you scroll down, you can see how the pattern looks repeated, as it would be in a knitted garment. If a user likes the pattern, links to yarns with the same colors are provided. Malloc - C

Along with a partner, I recreated the malloc, free, and realloc routines used in C. In our implementation, we used explicit free lists that were organized into bins of a certain size. This increases efficiency, because only bins of a similar size need to be searched.
